# 🧩 User Funnel Analysis

This project analyzes a simulated user funnel for an e-commerce product to identify drop-off stages and suggest improvements.

## 📊 Funnel Stages
1. Visit
2. Sign Up
3. Add to Cart
4. Initiate Checkout
5. Purchase

## 🔧 Tools Used
- Python (Pandas, matplotlib)
- SQL (for basic queries – optional)
- Jupyter Notebook

## 📈 Key Insights
- 70% of users signed up
- 50% of signed-up users added to cart
- Major drop-off between "Add to Cart" and "Initiate Checkout"
- Only ~12% of users completed purchase

## 💡 Recommendations
- Test a simplified checkout experience
- Add trust signals or UX improvements at payment step
- Trigger reminder emails after cart abandonment

## 📁 Files
- `/data/funnel_data.csv`: Synthetic dataset
- `/notebooks/funnel_analysis.ipynb`: Code & charts
- `/images`: Optional plots (e.g., funnel chart)
